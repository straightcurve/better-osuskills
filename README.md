# better-osuskills
gets results from the osu!Skills training page and PMs them on osu! using your own IRC credentials

# setup
```bash
git clone https://github.com/straightcurve/better-osuskills.git
cd better-osuskills
npm i
```

# start
```bash
npm run start
```

# package for windows and linux (64-bit)
```bash
npm run build
```
